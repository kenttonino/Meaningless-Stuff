## $\textnormal{Relational Operators}$

> - There are six relational operators to form Boolean expression.

```plaintext
<   : Less than.
<=  : Less than or equal to.
>   : Greater than.
>=  : Greater than or equal to.
==  : Equal to.
!=  : Not equal to.
```

```c
// Example 1.
#include <stdio.h>

int main() {
    int num = 41;
    num += 1;
    if (num == 42) {
        printf("You won!");
    }
}
```
