## $\textnormal{Variables}$

> - A `variable` is a name for an area in memory.

> - `Identifier` is the name of the variable.

```c
// Example 1.
int my_var;
my_var = 42;

// Example 2.
int my_var = 42;
```

```c
// Example 3.
#include <stdio.h>

int main() {
    int a, b;
    float salary = 56.23;
    char letter = 'Z';
    a = 8;
    b = 34;
    int c = a+b;

    printf("%d \n", c);
    printf("%f \n", salary);
    printf("%c \n", letter);

    return 0;
}
```
