## $\textnormal{Two-Dimensional \ Arrays}$

> - A `two-dimensional array` declaration indicates the number of <br />
    number of rows and the number of columns.

```c
// Example 1.
int a[2][3]; /* A 2 x 3 array */

// Example 2.
int a[2][3] = {
  {3, 2, 6},
  {4, 5, 20}
};

// Example 3.
int a[2][3] = { {3, 2, 6}, {4, 5, 20} };
```
